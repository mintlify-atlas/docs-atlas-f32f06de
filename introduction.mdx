## Welcome to shadcn-pricing-page

A beautiful, production-ready pricing component that's easy to integrate into your Next.js application. Built with modern technologies and best practices, this component provides everything you need to create stunning pricing pages.

## Key Features

<CardGroup cols={2}>
  <Card title="Animated Number Transitions" icon="wand-magic-sparkles">
    Smooth, eye-catching price animations using @number-flow/react with currency formatting and locale support.
  </Card>
  <Card title="Responsive Design" icon="mobile">
    Mobile-first design that adapts beautifully to any screen size, from phones to large displays.
  </Card>
  <Card title="Payment Frequency Toggle" icon="toggle-on">
    Interactive monthly/yearly switcher with animated transitions using motion (Framer Motion).
  </Card>
  <Card title="Flexible Tier System" icon="layer-group">

  </Card>
  <Card title="Dark Mode Support" icon="moon">

  </Card>
  <Card title="shadcn/ui Components" icon="puzzle-piece">

  </Card>

## What's Included

This package provides a complete pricing component system:

- **Pricing Component**: Main container that orchestrates the entire pricing section
- **PricingHeader**: Customizable header with title, subtitle, and frequency toggle
- **PricingCard**: Individual pricing tier cards with features, pricing, and CTAs
- **Configuration System**: Type-safe pricing tier configuration with full TypeScript support
- **UI Components**: Pre-built shadcn/ui components (Button, Badge, Tab, etc.)
- **Animations**: Smooth transitions and micro-interactions

## Tech Stack

Built with the latest and greatest:

- **Next.js 15** - React framework with App Router
- **React 19** - Latest React features and improvements
- **TypeScript 5** - Type safety and developer experience
- **Tailwind CSS 3.4** - Utility-first CSS framework
- **shadcn/ui** - High-quality, accessible UI components
- **@number-flow/react** - Animated number transitions
- **motion** - Production-ready animation library (Framer Motion)
- **next-themes** - Perfect dark mode support
- **lucide-react** - Beautiful, consistent icons
